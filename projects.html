Mavle (Student Education Platform)

Im Modul "Software Entwicklung & Programmierung" habe ich zusammen mit vier Gruppenmitgliedern eine Lehrplattform entwickelt, ähnlich zu Moodle. Zu meinen Hauptaufgaben gehörten das Schaffen einer Client-Server-Architektur, die Registrierung und die Kalenderfunktion.

Der Server wurde mittels Spring Boot erstellt und als Datenbank haben wir uns für MySQL entschieden. Für das Frontend nutzten wir JavaFX. Durch unsere Bemühungen haben wir es von über 20 Gruppen in die Top 3 geschafft.

Fahrverstoß Ermittlung

Für das Bachelorprojekt habe ich mich für "Vision-based Traffic Surveillance" entschieden und mittels Computer Vision diverse Fahrverstöße erkannt und ausgewertet. Neben einer Fahrbahnerkennung zur Ermittlung von Mindestabstandverstößen habe ich an einer Ampelphasenerkennung, Blinkererkennung sowie einer Kennzeichenerkennung und -auslesung gerarbeitet.

Zur Erkennung von Kennzeichen habe ich ein eigenes neuronales Netz trainiert, mit Hilfe von YOLOv4. Das Programm wurde in Python geschrieben und als Bibliotheken kamen hauptsächlich OpenCV, SciMath und Matplotlib zum Einsatz.

Point Cloud Streaming System

Im Rahmen meiner Bachelorarbeit habe ich ein Point Cloud Streaming System entworfen, welches die Point Cloud Objekte durch die Nutzung von DASH effizient überträgt.

Das entwickelte Programm wurde in Python geschrieben und passt die Qualität des Streams an die Bandbreite des Nutzers an, mit dem Ziel, eine kontinuierliche Wiedergabe zu gewährleisten.

Guse

Dieses private Projekt diente ursprünglich zum Lernen von Angular, um meine Arbeiten meiner Werkstudententätigkeit besser zu gestalten. Zur gleichen Zeit konnte ich meine TypeScript Kenntnisse auffrischen und mit der Nutzung von SASS mehr Kontrolle über die Darstellung der Webseite erlangen.

Guse hat das Ziel, den eigenen Tagesablauf zu planen und bietet durch seine einfache Oberfläche einen unkomplizierten Weg, um Tagesaktivitäten schnell zum Tag hinzufügen zu können. Für das Backend habe ich mich für einen NodeJS Server entschieden und die Daten werden in einer MongoDB Datenbank gespeichert.

k-anki

Der Kindle E-Reader von Amazon bietet seinen Lesern die Möglichkeit, eine Definition und Übersetzung von Wörtern anzuzeigen. Mit k-anki werden die nachgeschlagenen Wörter erfasst und mittels der Merriam-Webster und DeepL API in ein für Anki verständliches Format gewandelt.

Dieses Skript wurde in Python geschrieben und wertet den Inhalt der internen Vokabeldatei aus. Die Bedienung erfolgt über die Konsole, kann aber so konfiguert werden, dass die Konvertierung automatisch geschieht.

Path

Um mich mit dem Frontend Framework React vertraut zu machen und gleichzeitig die Lerninhalte der Universität zu strukturieren, habe ich Path entwickelt. Anstelle von JavaScript habe ich mich für TypeScript entschieden, weil es bei meiner Werkstudententätigkeit vermehrt zum Einsatz kam.

Ich nutze Path indem ich Fragen zum Lerninhalt aufschreibe und mit dem "Review" Modus die Schwierigkeit einer Frage beurteile. Durch Spaced repetition werden Fragen die mir schwerer fallen öfters angezeigt, als Fragen die ich mit Leichtigkeit beantworten kann.

Raspberry Pi Roboter

Bei diesem Projekt habe ich einen kleinen Roboter gebaut, welcher aus zwei Servos, einem Servocontroller, einer Kamera und etwas Pappe besteht. Gesteuert wird er durch ein Raspberry Pi.

Im obigen Beispiel habe ich OpenCV genutzt um mir die Pixelposition des Stiftes ausgeben zu lassen. Im nächsten Schritt stelle ich den Drehwinkel der blauen Servos so ein, sodass der Stift immer mittig von der Kamera aus steht.
